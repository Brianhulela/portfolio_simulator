# Portfolio Simulation: Decision-Making Systems Comparison

This project simulates and compares two simple algorithmic decision-making systems managing a $100 portfolio using historical US stock data.

The goal is to study how different allocation rules behave under identical market conditions, rather than to predict markets or build production trading systems.

Two strategies are tested:
- A momentum-based system that allocates capital based on recent price performance adjusted for volatility
- An equal-weight system that distributes capital evenly across all assets

Both systems operate within the same simulation environment, which includes transaction fees, portfolio rebalancing, and position constraints to reflect more realistic trading conditions.

The experiment focuses on understanding differences in:
- portfolio growth over time
- risk and volatility
- drawdowns
- risk-adjusted performance metrics

The results help illustrate how simple decision rules can lead to significantly different outcomes even when starting from the same capital and market data.
